Air-Bnb

Engineered a fully functional cross-platform rental marketplace application mirroring core Airbnb functionalities using React Native and TypeScript. The application leverages Firebase for secure authentication and real-time Cloud Firestore database management to ensure seamless synchronization of property listings and user reservations. I architected a robust global state management system using Redux Toolkit to efficiently handle complex booking logic, user sessions, and "favorites" lists. Key features include a high-performance virtualized list for browsing properties, advanced search filtering, interactive map integration, and a dynamic date-range selection engine, demonstrating a strong command of scalable mobile architecture and modern UI/UX replication.
